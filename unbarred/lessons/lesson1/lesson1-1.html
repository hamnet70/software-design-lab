HTML

Hypertext Markup Language (or HTML) is the language used to code webpages. Coders "mark up" the text of their pages with instructions for the web browser built in on how to display it. When a web browser, like Google Chrome or Internet Explorer, opens a page, it reads those those instructions that have been coded into the text.
